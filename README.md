# 💻 KonataVPN Desktop Client

Официальный десктопный клиент для подключения к скоростной сети **KonataVPN** на базе ядра **mihomo** (Clash.Meta).

## ⬇ Доступные файлы

В данном репозитории опубликованы готовые исполняемые файлы для Windows (x64):

1. **[installer-konata-vpn.exe](installer-konata-vpn.exe)** — Установщик приложения. Устанавливает клиент в папку `C:\KonataVPN` по умолчанию, создаёт ярлыки на рабочем столе и в меню Пуск.
2. **[konatavpn.exe](konatavpn.exe)** — Портативная (Portable) версия. Запуск в один клик без установки.
3. **[konata-mihomo.exe](konata-mihomo.exe)** — Standalone консольное ядро mihomo для продвинутых пользователей.

---

## ⚡ Особенности клиента

* 🟢 **Подключение в 1 клик**: Импорт подписки прямо с сайта профиля с помощью встроенной кнопки импорта через Deep Link протокол (`konatavpn://`).
* 🛠 **Два режима работы**:
  * **TUN-режим**: Полное туннелирование всего трафика операционной системы через виртуальную сетевую карту.
  * **Системный прокси**: Настройка системных параметров проксирования для браузеров и приложений.
* 📦 **Сворачивание в трей**: Удобное управление подключением из панели уведомлений Windows (системный трей).
* 🚀 **Автозапуск**: Автоматический старт приложения при включении компьютера.
* 🌍 **Умный выбор серверов**: Автоматический тест пинга и переключение на самый быстрый сервер.

## ⚙️ Установка и запуск

### Вариант 1: Установка (Рекомендуется)
1. Скачайте **`installer-konata-vpn.exe`**.
2. Запустите установщик (потребуются права администратора).
3. Следуйте инструкциям мастера установки.
4. Откройте приложение, перейдите в личный кабинет на сайте [konata.app](https://konata.app) и нажмите **«⚡ Импортировать в KonataVPN»**.

### Вариант 2: Портативный запуск
1. Скачайте **`konatavpn.exe`** и **`konata-mihomo.exe`**.
2. Положите их в одну папку (например, `C:\KonataVPN`).
3. Запустите `konatavpn.exe`.
